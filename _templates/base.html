{# Override Furo's base template to fix CHM/JavaScript errors in restricted hosts. Furo's ``body`` block contains only the theme bootstrap script: ``document.body.dataset.theme = localStorage.getItem("theme")``. That fails when ``localStorage`` or ``dataset`` is unavailable (CHM viewer, legacy WebView). Page layout and content come from ``page.html``, which calls ``{{ super() }}`` and then renders the full UI — so we replace the parent ``body`` block here without calling ``super()``. #} {% extends "!base.html" %} {% block body %} {% endblock %}
