Edition Comparison
==================

rsyslog Windows Agent is available in multiple editions. The manual describes
the full feature set, while edition-specific licensing determines which
features are available in a given deployment.

For Event Log Monitor deployments, licensing follows the same model as
EventReporter. A license is required for each source system whose Windows
Event Logs are collected or forwarded. This applies to physical systems and
virtual machines alike.

Use the official edition comparison page to confirm which services, actions,
and capabilities are included in the edition you plan to deploy:

`rsyslog Windows Agent edition comparison <https://www.rsyslog.com/windows-agent/edition-comparison/>`_
