EventReporter product Event ID reference
========================================

.. only:: event_id_catalog_preview

   .. warning::

      This appendix is an engineering preview. Its Event ID descriptions and
      troubleshooting guidance are still under review and are not yet published
      as part of the released product documentation.

Use this appendix to look up Windows Application Event Log entries generated by EventReporter.
Each Event ID links to its meaning, immediate checks, exact procedures, and escalation evidence.

Scope
-----

This reference covers product service diagnostics emitted by current supported builds.
It does not cover Event IDs collected from other Windows applications or IDs chosen
by an administrator in the Write to Windows Event Log action.
