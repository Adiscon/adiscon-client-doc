Edition Comparison
==================

Use the EventReporter edition comparison page when you need to confirm which
services, actions, and product capabilities are available in each edition.

For licensing scope, count each source system whose Windows Event Logs are
collected or forwarded through Event Log Monitor. This applies to physical
systems and virtual machines alike.

`EventReporter edition comparison <https://www.eventreporter.com/edition-comparision/>`_

**2026 major:** Licensing uses License V2 (``license.alic``) on 2026 builds.
Legacy keys and V2 files are not interchangeable across major versions. See
:ref:`license-v2`.
