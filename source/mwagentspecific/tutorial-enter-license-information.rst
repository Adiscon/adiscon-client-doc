Tutorial: Enter License Information
===================================

Goal
----

Enter the delivered license information into MonitorWare Agent and activate it
for the running service.

Prerequisites
-------------

- You received the registration name and license key from Adiscon.
- The MonitorWare Agent Configuration Client is installed.

Steps
-----

1. Open the MonitorWare Agent Configuration Client.
2. Select **General** -> **License**.
3. Enter the registration name exactly as delivered.
4. Use **Import from Clipboard** to paste the full license key.
5. Save and apply the configuration.
6. Restart the MonitorWare Agent service if required in your environment.

Verification
------------

Confirm that the license page no longer reports an invalid key state.
