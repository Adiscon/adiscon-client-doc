Tutorial: Export Configuration and Create a Debug Log
=====================================================

Goal
----

Export the current MonitorWare Agent configuration and capture debug logging
for troubleshooting or support.

Prerequisites
-------------

- The MonitorWare Agent Configuration Client is available.
- You know where to store the exported files.

Steps
-----

1. Open the MonitorWare Agent Configuration Client.
2. Export the current configuration to a file.
3. Open the debug options and enable debug logging.
4. Reproduce the issue you want to investigate.
5. Disable debug logging again once you have captured enough data.
6. Collect both the exported configuration and the debug log.

Verification
------------

Confirm that the exported configuration file and the debug log file both exist
and contain current data.
