Introduction
============

MonitorWare Agent is a Windows monitoring and log-processing platform for
collecting local and remote data, evaluating it with rules, and then storing,
forwarding, or acting on the result.

It is the most feature-complete member of Adiscon's Windows monitoring product
line. In one deployment, it can combine Windows Event Log collection, syslog
and SETP reception, file and service monitoring, active probes, filtering,
and alerting.

Use :doc:`understand-the-components` and the rest of :doc:`index` for the
active getting-started path.
