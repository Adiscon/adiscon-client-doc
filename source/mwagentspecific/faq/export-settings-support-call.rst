How Do I Export Settings and Debug Data for a Support Case?
===========================================================

Answer
------

Export the current configuration, capture a debug log while reproducing the
problem, and provide both artifacts together with a short problem description.

Details
-------

Support cases are easier to resolve when configuration and runtime evidence are
available together. The most useful combination is:

- exported configuration
- debug log covering the problem window
- the expected result and the actual result

Action path
-----------

1. Export the current configuration.
2. Enable debug logging.
3. Reproduce the problem.
4. Disable debug logging after the problem is captured.
5. Submit the exported configuration and debug log with the support request.
