How Do I Get a Printable MonitorWare Agent Manual?
==================================================

Answer
------

Use the PDF build of the manual when you need a printable or offline copy.

Details
-------

The HTML manual is the primary format for navigation and search. The PDF build
is the better choice when you need a single printable document.

Action path
-----------

1. Open the current MonitorWare Agent manual download area.
2. Download the PDF version of the manual.
3. Print or archive that PDF as needed.
