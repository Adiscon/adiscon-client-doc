Can I use an older product version with my license?
===================================================

Question
--------

Can I use an older Adiscon product version with my current license?

Answer
------

Yes. Your license can cover older product versions.

If the older product version does not support license files, contact Adiscon
support or sales and request a free license key for that product version.

Details
-------

Current products use a ``license.alic`` license file. Older products before
version 26 use a registration name and license key instead. Tell Adiscon which
product and version you need to run so the license can be issued in the right
format.
